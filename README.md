# Howling-Abyss_discordBot

리그 오브 레전드 칼바람 나락 맵에서   
챔피언들의 룬, 스펠, 아이템 및 스킬 빌드를 보여주는 디코 봇입니다.

---

현재 기능 :

* 챔피언 룬, 스펠 출력(텍스트 형식)
* 룬, 스펠 이미지 url 가져오기(텍스트 형식)

추후 구현 목록 :

* 아이템 및 스킬 빌드 추가
* 디스코드 봇 연동
* 이미지 출력

---

칼바람 챔피언 정보는 OP.GG에서 크롤링해옵니다.
